  1. Instructions

Please read the instruction carefully before filling the Application Form :


All fields marked * are compulsory.

  1. Keep the following information ready before filling the Application.
    1. Aadhar number, PAN number
    2. Copy of Aadhar Card and Pan Card
  2. check your Application thoroughly before Submitting
  3. You cannot edit the Application once it is submitted
  4. All information should be correct
  5. In case any information published by you is found incorrect you will be liable for disciplinary action by the department
  6. On Submit you can check the application status
  7. After submitting application it is verified by HR department
  8. Once application is verified by HR department then application is forwarded to Quarter Admin where quarter is allotted according to availibility



For more information please contact the concerned authorities
